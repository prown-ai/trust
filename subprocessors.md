# Subprocessors (Prown)

**Effective date:** 2025-12-16

Prown uses third-party subprocessors to provide and operate the Service. This page describes the subprocessors that may process personal data.

For questions, contact: **marcossponton@gmail.com**

## Current subprocessors

| Subprocessor | Purpose | Data categories | Notes |
|---|---|---|---|
| OpenAI | LLM API processing | Interview text, prompts/outputs, metadata | API usage for model inference |
| Google (Gemini) | LLM API processing | Interview text, prompts/outputs, metadata | API usage for model inference |
| MongoDB Atlas | Database hosting | Transcripts, debriefs, account/tenant metadata | Primary data store |
| Langfuse Cloud | Observability & evaluation | Prompt/response traces, evaluation metadata | Used to evaluate quality/cost/latency |
| Render | Application hosting | Application data in transit, logs/metadata | Hosting provider |
| OAuth providers (Google / Microsoft / LinkedIn) | Authentication | Identity and auth tokens | Used for sign-in |

## Updates

We may update this list from time to time. Material changes will be reflected in commit history and effective date.
